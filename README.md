# ckill
Ultra fast process killer for linux system. Very useful to contain some forking torjan.

__WARNNING__
- Use with caution
- DO NOT USE WITH ROOT

# Usage

```sh
gcc ./ckill.c -o ckill
./ckill procname1 procname2 ...
```
